# Encoding

This repository contains encoders and decoders for [goa](http://goa.design).

Use the `Consumes` and `Produces` DSL in your design to specify the encoding supported by your
services.

See the [corresponding documentation](http://goa.design/reference/goa/design/apidsl.html#func-consumes-a-name-apidsl-consumes-a:aab4f9d6f98ed71f45bd470427dde2a7).
